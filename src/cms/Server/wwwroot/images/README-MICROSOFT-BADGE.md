# Microsoft for Startups Badge

## Badge Placement

The Microsoft for Startups badge should be placed in this directory as:
`microsoft-for-startups-badge.png`

## How to Get the Badge

1. Visit the Microsoft for Startups portal: https://www.microsoft.com/startups
2. Download the official badge from your startup dashboard
3. Or use the guidelines at: https://www.microsoft.com/en-us/startups/blog/how-to-announce-your-startup-has-joined-microsoft-for-startups/

## Badge Requirements

According to Microsoft guidelines:
- Use the official badge only
- Include proper attribution
- Link to Microsoft for Startups when appropriate
- Follow brand guidelines for sizing and placement

## Current Usage

The badge is currently used on:
- Homepage (/)
- Login page (/login)
- Register page (/register)
- Onboarding flow (/onboarding)
