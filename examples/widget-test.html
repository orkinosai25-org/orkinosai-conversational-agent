📦 How to Embed on Your Website

To add this chat widget to your own website, simply include this HTML snippet:

<!-- Add this where you want the chat button to appear -->
<iframe
  src="http://localhost:5000"
  width="100%"
  height="700px"
  style="border: none; border-radius: 10px;"
></iframe>

Or open it as a full-page chat interface by navigating to http://localhost:5000

⚙️ Configuration

The widget supports both demo mode and production mode:

  • Demo Mode: Works immediately without any setup. Perfect for testing and development.
  • Production Mode: Configure Azure OpenAI credentials in appsettings.json for full AI capabilities.
  • Customization: Adjust temperature, max tokens, and system prompt in the configuration.
