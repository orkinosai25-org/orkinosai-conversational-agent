📦 Integration Code

To embed this widget on your website, use one of these methods:

Method 1: IFrame Embed

<iframe
  src="http://localhost:5000"
  width="100%"
  height="600px"
  style="border: none; border-radius: 12px;"
></iframe>

Method 2: Popup/Overlay

Use JavaScript to open the chat in a modal or fixed overlay:

function openChat() {
  window.open('http://localhost:5000',
    'sitechat-chat',
    'width=1200,height=800');
}

Method 3: Full Page

Link directly to the chat interface:

<a href="http://localhost:5000">
  Chat with SiteChat Agent
</a>

🚀 Getting Started

To run SiteChat Agent on your own server:

1. Install Dependencies

pip install -r requirements.txt

2. Start the Server

python main.py

3. Open in Browser

Navigate to http://localhost:5000

The application works immediately in demo mode. To enable full Azure OpenAI capabilities, configure your credentials in appsettings.json.

⚙️ Configuration Options

Customize the chat widget behavior:

  • Temperature: Control response creativity (0.0 - 1.0)
  • Max Tokens: Set maximum response length
  • System Prompt: Define the AI's personality and role
  • History Length: Number of past messages to remember
  • CORS Origins: Whitelist domains for widget embedding

All settings can be configured in appsettings.json or via environment variables.
